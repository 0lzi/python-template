# Python-Template

Personal repo for templating python projects

## Usage

### Set up virtual env

```
python3 -m venv ~/.{directory}
source ~/.{directory}
```

### Install requirements

` make install `

### Test 

` make test `

### Lint

` make lint `

### Format 

` make format `
